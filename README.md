## Quick Start

1. **Clone and install:**
   ```bash
   git clone https://github.com/davedonnellydev/react-typescript-starter.git
   cd react-typescript-starter
   npm install
   ```

2. **Set up environment variables:**
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your values
   ```

3. **Start development:**
   ```bash
   npm start
   ```

4. **Run tests:**
   ```bash
   npm test
   ```
